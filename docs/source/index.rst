################################
OpenPIV-Python-cxx Documentation
################################

OpenPIV-Python-cxx consists of Python and c++ modules for scipting and executing the analysis of a set of partivle image velocimetry (PIV) image pairs. Additionally, preprocessing and post-processing modules are included to help aid users in extracting as much information as possible from PIV images.
