openpiv_cxx.tools.uniform_scaling¶

openpiv_cxx.tools.uniform_scaling(x: ~numpy.ndarray, y: ~numpy.ndarray, u: ~numpy.ndarray, v: ~numpy.ndarray, scaling_factor: float) -> (<class 'numpy.ndarray'>, <class 'numpy.ndarray'>, <class 'numpy.ndarray'>, <class 'numpy.ndarray'>)¶

Apply an uniform scaling.

Parameters:

  • x (ndarray) – 2D arrays of x/y coordinates.

  • y (ndarray) – 2D arrays of x/y coordinates.

  • u (ndarray) – 2D arrays of u/v components.

  • v (ndarray) – 2D arrays of u/v components.

  • scaling_factor (float) – The image scaling factor in pixels per meter.

Returns:

  • x, y (ndarray) – 2D arrays of scaled x/y coordinates.

  • u, v (ndarray) – 2D arrays of scaled u/v components.
